# MyApplication

Este es un proyecto de Android hecho con Kotlin. Actualmente es una **app de ejemplo** con una sola pantalla que contiene:

- Un **EditText** con texto por defecto.
- Un **TextView** para mostrar información.
- Un **Button** de prueba (aún sin funcionalidad).

## Tecnologías

- Kotlin
- Android Studio
- ConstraintLayout
- Material Design Components
